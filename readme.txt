📄 My Portfolio
A professional digital portfolio website created using Python and Streamlit.
This project serves as an online platform to showcase my skills, projects, resume, and contact information.

🔥 Features Included
✅ Home / About Me Page
✅ Projects Showcase (with descriptions, technologies used, and GitHub links)
✅ Resume Section (Download and View Inline)
✅ Contact Form (Saves messages to CSV)
✅ Clean and Minimal UI with Sidebar Navigation

🚀 How to Run Locally

Install required dependencies:
pip install -r requirements.txt

Run the Streamlit app:
streamlit run app.py
